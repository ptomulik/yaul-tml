Description

Returns one of its two arguments T1, T2, dependieng on the value of integral constant c. if_c<c,t2,t2>type is a shortcut notation for if_<bool_<c>,t1,t2>type.

Expression semantics

For any Integral Constant c and arbitrary types t1 and t2:

typedef if_c<c,t1,t2>::type t;
  • Return type: Any type.
  • Semantics: Equivalent to
    typedef if_<bool_<c>,t1,t2>::type t;
Example
#include <yaul/tml/if.hpp>
#include <yaul/tml/bool.hpp>
#include <yaul/tml/is_same.hpp>
using namespace yaul::tml;
static_assert(is_same<if_<true_,char,long>::type, char>::value, "");
static_assert(is_same<if_<false_,char,long>::type, long>::value, "");
