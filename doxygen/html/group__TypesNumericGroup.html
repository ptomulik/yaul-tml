Description

A boolean Integral Constant wrapper.

Model of

Integral Constant.

Expression semantics

The semantics of an expression are defined only where they differ from, or are not defined in Integral Constant.

For arbitary integral constant c:

bool_<c>
  • Return type: Integral Constant.
  • Semantics: An Integral Constant x such that x::value = c and x::value_type is identical to bool.
Example
#include <yaul/tml/bool.hpp>
using namespace yaul::tml;
typedef bool_<false> f;
#include <yaul/tml/is_same.hpp>
static_assert( is_same<f::value_type, bool>::value, "");
static_assert( is_same<f::type, f>::value, "");
static_assert( f::value == false, "");
