Description

A generic Integral Constant wrapper.

Model of

Integral Constant

Expression semantics

The semantics of and expression are defined only where they differ from, or are not defined in Integral Constant.

For arbitrary integral type t and integral constant n:

typedef integral_c<t,n> x;
  • Return type: Integral Constant
  • Semantics: x is an Integral Constant such that x::value == n and x::value_type is identical to t.
Example
#include <yaul/tml/integral_c.hpp>
#include <yaul/tml/integral_c_tag.hpp>
using namespace yaul::tml;
typedef integral_c<short,8> eight;
#include <yaul/tml/is_same.hpp>
static_assert( is_same<eight::value_type, short>::value, "");
static_assert( is_same<eight::type, eight>::value, "");
static_assert( eight::value == 8, "");
static_assert( is_same<eight::numeric_tag, integral_c_tag>::value, "");
