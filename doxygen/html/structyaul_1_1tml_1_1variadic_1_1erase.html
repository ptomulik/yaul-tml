Description

erase performs a removal of one or more adjacent elements in the sequence with the guaranteed complexity O(1) and invokes F with the remaining arguments.

Expression semantics

For any Lambda Expression f with arity >= Last::value - First::value, an Integral Constants m1 and m2 with m2::value >=0 and m2::value > m1::value, and any types a0,...,a{n-1} (n >= m2)

typedef variadic::erase<f,m1,m2>::apply<a0,...,an>::type r;
  • Return type: Unspecified type
  • Semantics: equivalent to
    typedef apply<f,a0,...a{m1-1},a{m2},...an>::type r;
Example
#include <yaul/tml/variadic/erase.hpp>
#include <yaul/tml/quote.hpp>
#include <yaul/tml/int.hpp>
using namespace yaul::tml;
template <class... Args> struct seq;
typedef variadic::erase<quote<seq>, int_<1>, int_<3> >::apply<char, short, int, float>::type r1;
typedef variadic::erase_c<quote<seq>, 1, 3>::apply<char, short, int, float>::type r2;
#include <yaul/tml/is_same.hpp>
static_assert(is_same<r1,seq<char,float> >::value, "");
static_assert(is_same<r2,seq<char,float> >::value, "");
