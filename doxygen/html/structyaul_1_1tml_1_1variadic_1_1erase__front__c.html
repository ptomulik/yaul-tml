Description

erase_front_c removes first N arguments from the argument pack Args... with the guaranteed complexity O(1) and invokes F with the remaining arguments.

Expression semantics

For an n-ary Lambda Expression f, an integral constant m and any types a0,...,a{m+n}

typedef variadic::erase_front_c<f,m>::apply<a0,...,a{m+n}>::type r;
  • Return type: Unspecified type
  • Semantics: equivalent to
    typedef erase_front<f,long_<m> >::apply<a0,...,a{m+n}>::type r;
Example
#include <yaul/tml/variadic/erase_front.hpp>
#include <yaul/tml/quote.hpp>
#include <yaul/tml/int.hpp>
using namespace yaul::tml;
template <class... Args> struct seq;
typedef variadic::erase_front_c<quote<seq>, 2>::apply<char, short, int, float>::type r1;
typedef variadic::erase_front<quote<seq>, int_<2> >::apply<char, short, int, float>::type r2;
#include <yaul/tml/is_same.hpp>
static_assert(is_same<r1,seq<int,float> >::value, "");
static_assert(is_same<r2,seq<int,float> >::value, "");
